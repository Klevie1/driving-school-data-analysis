# Driving School Data Analysis Dashboard

## 📊 Overview
This project analyzes the financial performance of a driving school using Excel.

## 🎯 Objectives
- Analyze revenue and expenses
- Track student enrollment
- Identify business trends

## 🛠 Tools Used
- Microsoft Excel

## 📈 Key Metrics
- Total Revenue: GHS 570,700
- Total Profit: GHS 499,090
- Total Expenses: GHS 71,610

## 🔍 Key Insights
- Revenue peaked in October
- Fuel is the highest expense
- Student enrollment varies monthly
- Business maintains strong profit margins

## 💡 Recommendations
- Reduce fuel costs
- Increase marketing during low months
- Improve debt collection
